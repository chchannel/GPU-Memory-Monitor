# GPUメモリ監視プログラム 開発完了報告

GPUメモリの使用状況をリアルタイムに把握することに特化した、モダンでクリーンな監視ツールを開発しました。

## 実装された主要機能
- **超スムーズな操作性**: 重いデータ取得処理をバックグラウンドへ分離し、リストの更新を差分更新に最適化。
- **完全ステルス実行**: バックグラウンドでのデータ取得時もウィンドウ（PowerShell等）が一切表示されず、作業の邪魔をしません。
- **高精度リアルタイム監視**: 3秒ごとに情報を更新。Windows標準のパフォーマンスカウンターを利用することで、ブラウザや描画アプリのメモリ変動も100%確実に捉えます。
- **総使用量表示**: GPU全体のメモリ使用率（%）とGPU名、使用MB/合計MBを大きく表示。
- **あらゆるアプリを検知**: Firefox（YouTube再生）、Eagle、ゲームなど、`nvidia-smi` では 0MB と表示されがちなアプリも正確な数値で表示。
- **ソート機能**: 「使用量順」「名前順」の切り替えが可能。
- **フィルタリング**: 1MB未満の微小なプロセスを隠す機能（0MBを隠す）。
- **クリーンなUI**: PySide6によるグレーベースのモダンでコンパクトなデザイン。

## 技術スタック
- **GUI**: PySide6
- **Backend**: Python (Windows パフォーマンスカウンター / nvidia-smi 併用)
- **Architecture**: マルチスレッド非同期処理 (QThread)
